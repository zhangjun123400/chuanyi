07 · 状态与异常

PRD 验收所需的完整异常与业务规则展示

禁止提交

额度不足

提交生成前预扣额度，额度不足时禁止提交，并引导购买额度。

上传失败

文件超过20MB

图片过大，请压缩后重新上传。该状态不能继续生成。

可继续

模特姿态不适合

半身模特不适合连衣裙完整展示，允许用户强制继续。

部分失败

视频失败，图片成功

成功图片可下载，失败视频展示原因并返还视频部分额度。

全部失败

系统生成失败

因系统错误全部失败时自动返还额度，并允许重新生成。

待修复

衣摆异常

图片可进入局部修复。视频 MVP 暂不支持局部编辑。

不可下载

失败素材

失败结果不展示下载入口，只展示失败原因与重新生成。

复现参数

结果详情

结果页保留任务输入、参数、模型质量标签，便于复现。
