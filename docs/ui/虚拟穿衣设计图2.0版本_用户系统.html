📌 侧边栏用户区设计要点

  • 用户头像取邮箱首字母大写，背景色使用 brand-light
  • 角色以 badge 形式展示（admin=紫色 / operator=灰色）
  • 登录有效期以绿色圆点 + 文字展示，实时计算剩余天数
  • 「修改密码」触发弹窗（见 V2-09），「登出」清除令牌并跳转登录页
  • admin 角色额外显示「管理后台」入口链接（新标签页打开 localhost:3001）

📌 主应用登录页设计要点

  • 登录页背景使用深色渐变 + 装饰性半透明圆形光晕，与品牌调性一致
  • 与后台登录页（V2-01）共享认证接口，但视觉上区分定位：主应用强调「AI试穿工作台」，后台强调「管理」
  • 未勾选「记住我」时显示「7 天内保持登录」，勾选后变为「30 天内无需重新登录」
  • 被禁用用户有独立错误提示（黄色警告），与密码错误的红色提示区分
  • Operator 误入后台时提示「前往主应用工作台」而非直接拒绝
